# Stretto Web

React + TypeScript + Vite frontend for the Stretto application.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev          # Start dev server at http://localhost:5173
npm run test         # Run Vitest tests
npm run lint         # Lint with ESLint
npm run build        # Type-check + build for production
npm run generate     # Regenerate TypeScript API client (requires API running at localhost:5000)
```
